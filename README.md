## Description

**WP Delete Attributes** est un plugin WordPress conçu pour simplifier le nettoyage de vos contenus en éliminant les attributs HTML superflus. Il ajoute une action groupée "Supprimer les attributs HTML" permettant de nettoyer en masse les articles et les pages directement depuis la liste des contenus.

[![Voir le tutoriel](https://img.youtube.com/vi/hGkWy4QpdDk/maxresdefault.jpg)](https://www.youtube.com/watch?v=hGkWy4QpdDk&ab_channel=KevinBenabdelhak)


### Fonctionnalités principales :
- **Action groupée :** Supprimez facilement des attributs HTML en masse depuis la liste des articles et pages.
- **Exceptions configurables :** Définissez les attributs qui ne doivent pas être supprimés (ex. : href, src, alt, title).
- **Gestion des types de contenu :** Activez le plugin pour les types de contenu de votre choix (articles, pages, etc.).

## Installation

1. Téléchargez le fichier ZIP du plugin :
   Rendez-vous sur la page du plugin à l'adresse [WP Delete Attributes](https://kevin-benabdelhak.fr/plugins/wp-delete-attributes/) et téléchargez le fichier ZIP.

2. Uploader le plugin sur votre site WordPress :
   - Allez dans « Extensions » > « Ajouter »
   - Cliquez sur « Téléverser une extension »
   - Sélectionnez le fichier ZIP téléchargé puis cliquez sur « Installer maintenant »
   - Activez le plugin en cliquant sur « Activer ».

## Utilisation

1. Rendez-vous sur la page des options du plugin pour configurer les exceptions et les types de contenu.
2. Accédez à la liste de vos articles ou pages.
3. Sélectionnez les éléments sur lesquels vous souhaitez supprimer les attributs HTML.
4. Dans les actions groupées, choisissez "Supprimer les attributs HTML" et validez.

## FAQ

### Quels attributs HTML peuvent être supprimés ?
Tous les attributs qui ne sont pas spécifiés comme exceptions peuvent être supprimés. Vous pouvez configurer ces exceptions dans les options du plugin.

### Est-ce que ce plugin fonctionne avec tous les types de contenu ?
Oui, le plugin peut être activé pour divers types de contenu, y compris les articles et les pages.

### Les modifications sont-elles réversibles ?
Il est recommandé de faire une sauvegarde de votre contenu avant de procéder, car les attributs supprimés ne peuvent pas être récupérés via le plugin.
